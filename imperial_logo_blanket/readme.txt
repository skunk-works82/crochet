This project is a long lap blanket with the imperial logo

Finished dimensions:  36" x 72"

Yarn used:   Lion brand Heartland
     pattern idx  |  Yarn Color
          0           153 Black Canyon
          1           113 Redwood
          
stitch gauge:  12S / 15R
stitch type :  SC

File list:
-------------------------------------
pattern.svg         :  vector file containing design of blanket.  Layers include other varriants, including imperial text.
pattern.jpg         :  bitmap exported from pattern.svg used to generate stitch pattern
pattern_gen.m       :  matlab script file used to convert pattern.jpg into pattern
pattern.txt         :  pattern matrix showing stitches in row / column format
pattern_reduced.txt :  pattern reduced into a per-row stitch count per color.  Each row color block is formatted "count:color"
                           10:0 5:1 10:0   would translate to "10 stitches color 0 (black) .... 5 stitches color 1 (red) ... 10 stitches color 0"
